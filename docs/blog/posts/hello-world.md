## 接下来要做什么？

使用 Markdown 编写博客非常舒适。这篇博客文章存储在 `docs/blog/posts/hello-world.md` 之中。当你需要增加新的博客时，只需要在同一个目录下新建 `.md` 文件即可，系统会自动根据文件内的 `date`（日期）字段为你按照时间轴进行排序！

这是代码块演示：

```python
def greet(name: str):
    print(f"Hello, {name}!")

greet("World")
```
